# PicTour v4.6.3 — Checkout presencial modular ampliado

## Objetivo

Ajuste fino de interface para deixar o frame do checkout presencial modular mais largo na aba Venda Rápida.

## Mudanças

- O painel lateral do checkout presencial agora usa largura mínima maior em telas desktop.
- Em telas grandes, o painel pode expandir até aproximadamente 560px.
- Em telas menores, o layout continua quebrando para uma coluna para não gerar overflow.
- Mantida compatibilidade com pagamentos divididos, slots de produtos e monitor do cliente.

## Motivo

O checkout modular passou a ter mais informações: produtos adicionados, slots de foto, pagamentos divididos, troco e controles do monitor. O painel anterior ficava visualmente apertado para operação real de caixa.
